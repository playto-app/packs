# Slay the Spire — EN → JA pack (seed)

Covers the core glossary for playing Slay the Spire in English while learning Japanese equivalents.

## Summary

- **Game build**: English (Steam)
- **Source language**: English (`en`)
- **Target language**: Japanese (`ja`)
- **Glossary entries**: 66
- **Contributor**: director-sample (seed pack — open to improvements via PR)
- **Version**: v1 (2026-04-20)

## Capture settings

Default Cursor Follow preset (400 × 300, offset y=-250). Tuned for the in-game card tooltip box — hover a card and cursor mode captures the description. If the tooltip moves with game UI scaling, adjust W/H/X/Y in Settings → Capture Area.

## Known gaps / open follow-ups

- Character class exclusives (Silent-specific shiv/discard terms, Defect-specific orb types like Frost / Lightning / Dark / Plasma) are only lightly covered — PR welcome.
- Act-specific enemies (Cultist, Book of Stabbing, Writhing Mass, etc.) not yet translated.
- Uses the reading "休憩所" for Rest Site / Campfire to match the common community rendering — swap to "たき火" if the game's official JA localization differs.

## How to use this pack

1. Download `game-pack.playto-pack` from this folder.
2. In Playto, open the **Profile** tab → **Import pack** → select the file.
3. Playto shows a preview — choose **Import All** to copy capture settings + glossary, or **Glossary only** to keep your own capture setup.
